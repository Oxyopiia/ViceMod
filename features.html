Vice Features List

List last Updated: 11/05/24

Summary

Vice is an open-source Fabric utility mod specially designed for the DoomTowers server, featuring various helpers, muters, and features.

General
  • Backpack Renaming
  • Trash Protection
  • Hide Revolver Blindness
  • Improved Fishing Ding
  • Forge Timers UI
  • Player Stats UI
  • Worldguard Message Hiders
  • Modify/Mute the Volume of many Abilities
World-Specific

    World 4

  • Show Next Cooking Item
  • Cooking Stock UI
  • Cooking Order Tracker
  • Beef Cooking Timer

    • World 11

  • Train Timer UI

    • World 12

  • Evan Notification

    • Exonitas

  • Power Box Timer
  • Baby Mode Makes players have baby models to allow better visibility.
  • Hide Bloat Messages
Item Abilities
  • Item Cooldown Overlay Displays Item Cooldowns for hotbar items in real time with:
    - 5 Unique styles
    - Customisable opacity
    - Toggleable cooldown next to cursor
    - Toggleable ready text
  • Wrong Set Indicator
  • Hide Set Requirement Warnings
  • Customisable Cooldown Title
Bosses
  • Boss Counter
  • Despawn Timers
  • Despawn Warning

    • Specific

  • Wasteyard Cooldown Timer Renders a timer for the Wasteyard cooldown. Also sends a chat message and a customisable sound when the Wasteyard is ready.
  • Abyssal Vice Laser Warning
Expeditions
  • Run Overview
  • Double tap to Drop Valuable Items
  • Merchants Overlay
  • Room Waypoints
  • Defibrillator Counter
  • Hide Style Points
  • Auto Relay Information Sends information about newly discovered villagers to your teammates. This can also be filtered out even after opting out of sharing.
    Use at your Own Risk (probably fine tho)
Arenas
  • Live Arena Information Displays drop rates, wave timers, and more information in Arenas.
  • New Mob Effects Notification
  • Cooldown Reminder
